Introduction

This page contains artifacts for the paper: "OVERTON" submitted to the ACM CCS 2024. The artifacts include OVERTON source code, VEINS&F2MD simulator version used for experiments, traffic scenario implementations, and generated data for the scenario grid we detailed in the paper.

Running OVERTON framework

  • Instructions and code for the OVERTON framework is provided here.
  • VEINS simulation maps and scenario configurations are provided here.
  • Generated traffic logs for OVERTON evaluation is provided here.

    • VEINS Scenarios for Experiments

    Footages and resources for three different traffic scenarios used for experiments. Each scenario is tailored to demonstrate the capabilities of our solution under different traffic conditions and environments. Attacker nodes are depicted in red and genuine nodes in green.

    Manhattan Grid

    This scenario implements a single lane 10x10 grid map.

  • We have developed a grid-likemap with single lanes, referred as Manhattan Grid.
  • Manhattan grid consist of single lane roads with a 10x10 grid layout. Intersections of roads are spaced 100m apart.
  • We use 100 vehicles with varying attacker densities that implements an equally represented mixture of considered attacks.

    • Video Link: https://youtu.be/0aMBMmMEcPw

    Erlangen Flock

    This scenario implements a realistic rush hour scenario with frequent congestions.

  • A modified version of Erlangen Map, implements a more realistic traffic scenario in the city of Erlangen, Germany.
  • Specifically, 150 vehicles are flocking out of a parking lot with 6 second intervals, and head to another junction in a single designated route.
  • The vehicles are not allowed to detour for another route, which forces occasional stops and traffic jams during commute.

    • Video Link: https://youtu.be/AvogtLSoI4g

    IRTSX Map

    This scenario emulates regular traffic scenario in the IRTSX campus.

  • This is a relatively small network which provides fast and predictable results.
  • It implements traffic lights, pedestrians and crossroad junctions and organic network properties with realistic path loss models.
  • There are more than 300 entities that arbitrarily enter and leave the network.

    • Video Link: https://youtu.be/vBeQfSryIos

    VEINS&F2MD simulator

    OVERTON is implemented to work on existing message interface for both ITS-G5 (IEEE 802.11p) and C-V2X (3GPP PC5 Mode 4) communication standards. The evaluation of OVERTON is performed by VEINS&F2MD V2X traffic simulators. Instructions for installing the simulator is provided here.
