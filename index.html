Black Box Framework

Our scoring framework computes Privacy, Safety, and Utility from a paper-specific metrics file and a fixed weights file.

Config structure

{
  "Privacy": { "P1a": { "answer": true,  "weight": 0.1, "evidence": "" },
                "P1b": { "answer": false, "weight": 0.1, "evidence": "" } },
  "Safety":  { /* ... */ },
  "Utility": { /* ... */ }
}

Weights file

{
  "Privacy": { "P1a": 0.1, "P1b": 0.1, /* ... */ },
  "Safety":  { /* ... */ },
  "Utility": { /* ... */ }
}

The script awards each true metric the corresponding weight and sums to produce final scores.

How to Evaluate a Paper

  1. Fill the metrics JSON for the paper (see template above).
  2. Use the fixed weights JSON.
  3. Run the scoring script (see Scripts).
  4. Collect scores and export to CSV for the Results section.
# Example command
python tools/score_paper.py \
  --metrics configs/paper123.metrics.json \
  --weights configs/weights.json \
  --out outputs/paper123_scores.json

Scripts

Minimal commands to reproduce the main results.

Setup

source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt

Run

python scripts/run_experiments.py --config configs/default.json
python scripts/plot_results.py --input outputs/results.csv --out figs/
