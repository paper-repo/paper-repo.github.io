Introduction

In this study, a comprehensive systematization of knowledge (SoK) in the field of privacy in cyber-physical systems (CPS) and autonomous systems (AS) is presented, with a special focus on methods based on differential privacy. This page provides an overview of our proposed approach and the main results of our study.

Black Box Framework

We propose a framework that works as a black box, taking a research paper as input and producing three output scores for three key metrics: Privacy, Safety, and Utility.

Each metric is divided into several factors representing both technical depth and practical relevance. The scoring is additive, and each factor has a fixed weight, giving a final normalized value between [0, 1].

Inside the Black Box

A Python-based evaluation code is executed inside the black box. The code takes two JSON files as input:

  • Weight file :a predefined JSON file containing fixed weights for all evaluation factors, designed by us.
  • Paper file :a customizable JSON file filled by the evaluator, indicating which factors are true/false by the target paper.

When the code runs, it compares the evaluator’s inputs with the weight file. For every factor marked as true, its corresponding weight is added to the total score. The sum is a value between 0 and 1, representing the final scores for Privacy, Safety, and Utility the outputs of the black box.

We evaluated all references using this framework. The quantitative results are summarized in the table below and visualized in an interactive 3D plot. The plot was generated with Plotly and customized to highlight the relationship between Privacy, Utility, and Safety for each paper.
