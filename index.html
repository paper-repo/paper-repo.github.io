Introduction

In this study, a comprehensive systematization of knowledge (SoK) in the field of privacy in cyber-physical systems (CPS) and autonomous systems (AS) is presented, with a special focus on methods based on differential privacy.

Black Box Framework

we propose a framework that works as a black box, takes a research paper as input and produces three output scores for three metrics:privacy, safety, and utility. Each metric is divided into several factors that represent both technical depth and practical relevance. The scoring is additive, and each factor has a fixed weight, giving a final normalized value between [0,1]

How to Evaluate a Paper

The evaluation is based on predefined configuration JSON files. One configuration file must be completed by the user to indicate whether each factor is present in the paper (true/false). Another file contains the predefined weights for all factors. The framework compares these two files and calculates the final scores for privacy, utility, and safety.
